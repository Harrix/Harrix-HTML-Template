Типографика

В данном блоке пример использования обычного текста в несколько абзацев. Непосредственно в данном абзаце не будет применения никаких дополнительных украшательств текста. Просто несколько строк в абзаце. Поэтому мне надо о чем-нибудь тут написать, чтобы текста было побольше. Например, можно написать о том, что угол наклона оси вращения Земли составляет 23,44°. Именно такой угол наклона в моём логотипе.

А теперь начнем показывать некоторые возможности оформления текста. Например, можно показать сочетания клавиш с помощью такой конструкции Ctrl + V. Таким же образом можно оформлять и вызов пунктов меню: File → Open.

Для выделения текста используется тэг code. Например, название тэга именно так и оформлено. А это выделение обычным жирным.

Ссылки (harrix.org) отображаются другим цветом и подчеркиваются. При наведении на них цвет ссылки меняется на более светлый.

Списки оформляются вот таким вот образом:

  • Меркурий
  • Венера
  • Земля
    • Луна
  • Марс
    • Фобос
    • Деймос

А это абзац текста после списка. Он тут для того, чтобы можно было оценить расстояние от списка до последующего текста.

#include "stdafx.h"//Visual Studio может попросить
 
#include <iostream>
#include <fstream>
#include <string>
#include <sstream>
 
using namespace std;
 
int main()
{
 setlocale(LC_ALL, "RUSSIAN");//Чтобы русский текст поддерживался
 
 //Создаем файловый поток и связываем его с файлом
 ifstream file("input.txt");
 
 if (file.is_open())//Если открытие файла прошло успешно
 {
  cout << "Файл открыт." << endl;
 
  string line;//Строчка текста
 
  //Будем считывать информацию построчно до тех пор,
  //пока не закончится файл
 
  while (getline(file, line))
  {
   //cout << line << endl;//Можно посмотреть, что в строчке считалось
 
   //Теперь в line хранится содержимое строчки из файла.
   //Будем её разбирать на составные части.
 
   //В нашем файле идут две строчки, а потом два числа
   string name, surname, info;
   int age, money;
 
   //Создадим поток для считывания данных из строчки
   istringstream iss(line);
 
   string token;
   int i = 0;
   while (getline(iss, token, '\t')) {
    if (i == 0) surname = token;
    if (i == 1) name = token;
    if (i == 2) info = token;
    if (i == 3) age = atoi(token.c_str());//перевод string в int
    if (i == 4) money = atoi(token.c_str());//перевод string в int
    i++;
   }
 
   //Выведем наши данные
   cout << "Данные из строчки:" << endl;
   cout << "\tФамилия: " << surname << endl;
   cout << "\tИмя: " << name << endl;
   cout << "\tИнформация: " << info << endl;
   cout << "\tВозраст: " << age << endl;
   cout << "\tДеньги: " << money << endl;
  }
 }
 else
 {
  cout << "Не удалось открыть файл." << endl;
 }
 
 system("pause");
    return 0;
}
